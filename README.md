# Retail-Consumer-Behavior-Analysis
This project analyzes a leading retail company's consumer behavior dataset to identify trends, improve customer engagement, and optimize marketing strategies. The analysis explores how factors like demographics, discounts, reviews, and seasons drive purchasing decisions and customer loyalty.
